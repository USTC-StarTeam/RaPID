RaPID

1. Paper

Hongchao Gu, Dexun Li, Kuicai Dong, Hao Zhang, Hang Lv, Hao Wang, Defu Lian, Yong Liu, and Enhong Chen. RAPID: Efficient Retrieval-Augmented Long Text Generation with Writing Planning and Information Discovery. In Findings of ACL 2025, pages 16742-16763, Vienna, Austria, 2025.

RaPID combines retrieval-grounded outlines, attribute-constrained search, and plan-guided article generation for efficient long-form writing.

3. Quick Start

conda create -n rapid python=3.10
conda activate rapid
pip install -r requirements.txt
python example.py --retriever google --output-dir ./results --max-thread-num 3 \
  --do-clarify --do-research --do-generate-outline --do-generate-article \
  --do-topo-generation --do-polish-article --interface console
